# Recallify

Paste any text → AI generates flashcards → review with spaced repetition (SM-2).

Turn articles, notes, or transcripts into a study deck in seconds.

## Tech Stack

Next.js (App Router) · TypeScript · Drizzle ORM · PostgreSQL (Neon) ·
Clerk · Vercel AI SDK · Tailwind + shadcn/ui · Vitest · Playwright
